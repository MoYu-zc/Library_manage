# Library_manage
图书管理系统  java 

## 项目部署

该项目是`eclipse`版本

>
>
>记得`mysql-connector-java-5.1.25-bin.jar`、`poi-3.9.jar`的导入，给你们放在项目里了。
>
>创建好数据库，更改项目中数据库密码，sql文件也给你了
>
>

**项目功能：**

- 登录、注册
- 采购管理，添加新采购、历史采购查询
- 销售管理，添加新销售，查询
- 人员管理，供货商、销售商管理
- 账款管理，查询操作
- 报警管理，设置、查询
- 客服即时通讯
